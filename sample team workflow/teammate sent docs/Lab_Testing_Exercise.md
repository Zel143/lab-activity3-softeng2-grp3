# **Objectives**

1\. Identify system requirements  
2\. Create test cases

4\. Execute and evaluate testing results

# **Procedure**

1\. Select a project feature (e.g., login, registration)  
2\. Write at least 5 test cases  
3\. Convert test cases into test scripts  
4\. Execute test scripts and record results
